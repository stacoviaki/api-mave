## ✅ Pré-requisitos (o que ter instalado)
- **Golang** — https://go.dev/doc/install  
- **Docker** — (para o Postgres)  
- **Flyway** — https://www.red-gate.com/products/flyway/editions/  
- **Air** — para auto reload (opcional) — https://github.com/cosmtrek/air

## Rodar Projeto
- Dentro de cmd/api/main.go rode o comando: go run main.go
- não esqueça de subir o container docker lá está o postgresql
- não esquece de configurar o flyway/migrations
